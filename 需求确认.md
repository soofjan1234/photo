# 项目需求确认文档

## 1. 项目概述

**项目名称**：触摸大屏图片展示系统
**目标用户**：大屏展示场景下的观众
**应用环境**：触摸大屏连接Windows 11主机
**开发语言**：Python + PyQt5

**项目简介**：
开发一个适用于触摸大屏的图片展示软件，实现屏幕分区域独立滑动的相册功能，提供流畅的用户交互体验。

## 2. 功能需求

### 2.1 界面布局
- 屏幕分为3个独立区域，每个区域作为一个独立的相册
- 三个区域水平排列，占据整个屏幕空间
- 界面简洁，无干扰元素，突出图片内容展示

### 2.2 图片展示功能
- 每个区域完整展示单张图片，不裁剪，保证图片完整性
- 支持常见图片格式（JPG、PNG、BMP等）
- 图片自动适配显示区域，保持比例不变形

### 2.3 交互功能
- 支持触摸滑动操作：在每个区域内上下滑动切换图片
- 滑动必须流畅，不得在图片切换过程中出现卡顿或停留在半张图片状态
- 循环浏览：浏览到最后一张图片后，继续下滑会回到第一张图片
- 同时支持鼠标操作作为备用交互方式

### 2.4 配置功能
- 通过配置文件定义三个相册对应的图片文件夹路径
- 配置文件支持自定义相册名称
- 配置文件格式简单易编辑

## 3. 部署与维护

### 3.1 安装要求
- 提供可执行文件（.exe），无需安装Python环境
- 支持绿色部署，可直接运行
- 包含简单的安装说明文档

### 3.2 配置说明
- 提供配置文件模板和说明
- 详细说明如何修改相册路径和名称

### 3.3 维护支持
- 提供基础故障排查指南
- 支持通过配置文件调整基本参数，无需重新编译

## 4. 验收标准

1. 程序能在Windows 11环境下正常运行
2. 屏幕正确分为三个独立区域，每个区域可独立上下滑动
3. 图片加载和切换流畅，无明显卡顿
4. 滑动切换时不会停留在半张图片状态
5. 能从配置文件指定的文件夹正确加载图片
6. 支持循环浏览（最后一张下滑到第一张）
7. 触摸操作和鼠标操作均正常响应
8. 程序在长时间运行后仍然稳定
